[ 선수 사진 넣는 폴더 ]

■ 네이밍 규칙 (이것만 지키면 자동 표시)

   {국가코드}_{등번호}.{확장자}

   - 국가코드: kor(한국) / mex(멕시코) / cze(체코) / rsa(남아공)
   - 등번호:   players.json의 해당 선수 number
   - 확장자:   jpg, jpeg, png 중 하나

   예) 손흥민(한국, 7번)  ->  kor_7.jpg
       김민재(한국, 3번)  ->  kor_3.png
       시크(체코, 10번)   ->  cze_10.jpeg

■ 사용법
   1. 위 규칙대로 이름 붙인 이미지 파일을 이 폴더(public/players/)에 넣는다.
   2. 끝. players.json을 수정할 필요 없이 카드에 자동으로 사진이 나온다.
   3. 해당 파일이 없으면 자동으로 등번호 아바타가 표시된다.

■ 우선순위
   ① 여기 올린 규칙 파일  >  ② players.json의 photo URL(위키미디어 등)  >  ③ 등번호 아바타
   - 즉 규칙대로 올린 사진이 photo URL을 항상 덮어쓴다.
   - 같은 등번호의 파일은 jpg -> jpeg -> png 순서로 먼저 찾은 것을 사용.
   - 권장: 정사각형(1:1), 얼굴 중심 이미지.

* 현재 kor_7.jpg 는 규칙 시연용 샘플 이미지입니다. 실제 손흥민 사진으로 교체하세요.
